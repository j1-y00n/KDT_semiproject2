## 💻 Getting Started

### Installation
가상환경 생성
```python
  python -m venv venv
```
가상환경 활성화
```python
  source venv/bin/activate
```
django 설치
```python
  pip install django==3.2.18
```
gitclone  
의존성 파일 설치
```python
  pip install -r requirements.txt
```
### Develop Mode
서버실행
```python
  python manage.py runserver
```
